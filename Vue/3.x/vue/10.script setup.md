## 资源路径

> 使用`URL`构造器，便于构建工具分析资源。
>
> `import.meta.url`：表示当前模块的路径。

**单个资源分析**

```js
new URL('@/assets/Wallpaper1.jpg', import.meta.url);
```

**所有文件后缀分析**

```js
// 会将assets文件夹下的jpg文件都打包
new URL(`@/assets/${fileName}.jpg`, import.meta.url);
```
