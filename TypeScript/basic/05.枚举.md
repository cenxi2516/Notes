> 枚举分为：**数字枚举**、**字符串枚举**。
>
> - 枚举，在编译结果中被编译为一个对象。
> - 通常用于约束某个变量的取值范围。



## 数字枚举

> 1. 数字枚举值会自动自增。
> 1. 被数字枚举约束的变量，可以直接赋值数字。

```ts
enum Gender {
    Male, // 0
    Female // 1
}

let gender: Gender;

gender = Gender.Male;
```



## 字符串枚举

```ts
enum Color {
    RED = '#f00',
    GREEN = '#0f0',
    BLUE = '#00f'
}
```
